# Curso de Programação em Python

## 📌 Descrição
Curso de Programação em Python focado no desenvolvimento de aplicações práticas, abordando lógica de programação, algoritmos, estruturas da linguagem, manipulação de dados, arquivos, testes e noções de jogos 2D. Prioriza boas práticas, raciocínio lógico e resolução de problemas em projetos reais.

## 🎯 Objetivo 
Capacitar o aluno a desenvolver aplicações em Python utilizando boas práticas de programação, desde a análise do problema até a validação do software.

## 📚 Conteúdos Abordados
- Fundamentos da computação
- Lógica de programação e algoritmos
- Estruturas da linguagem Python
- Manipulação de dados e arquivos
- Funções e modularização
- Tratamento de exceções
- Estruturas de decisão e repetição
- Testes de software
- Introdução a jogos 2D (Pygame)

## 🛠️ Tecnologias
- Python
- Pygame

## 🚀 Metodologia
Aprendizado baseado em projetos práticos, com atividades individuais e colaborativas, estimulando o pensamento lógico e a autonomia.

## 📈 Resultado Esperado
Ao final do curso, o aluno será capaz de desenvolver, testar e validar aplicações em Python, aplicando boas práticas de programação.

---

💡 *Curso ideal para iniciantes que desejam entrar no mundo da programação.*
